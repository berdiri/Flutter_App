## 🎨 Tech Stack & Architecture

Aplikasi ini dirancang dengan fokus pada efisiensi memori, rendering UI yang mulus pada *refresh rate* tinggi, serta struktur kode yang modular:

* ⚡ **Flutter Engine** — Framework utama untuk performa native 60fps+ lintas platform.
* 🎯 **Dart Lang** — Optimalisasi kompilasi *Ahead-of-Time* (AOT) untuk startup app yang instan.
* 📱 **Cross-Platform Target** — Basis kode tunggal yang dioptimalkan khusus untuk ekosistem Android dan iOS.
